All About Me

I'm Evelyn, a first class computer science graduate and an avid programmer with experience in many areas such as machine learning, artificial intelligence, REST APIs and web development. I'm an aspiring full-stack developer and am currently working on a group project for the developers society here at NTU, creating a website that allows them to create events and monitor attendance. I'm currently working as a trainee developer at Next working with the ELK stack to monitor and analyse log data to keep the business running smoothly.
