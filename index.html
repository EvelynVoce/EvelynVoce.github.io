All About Me!

I'm Evelyn, a final year computer science student at Nottingham Trent University and an avid programmer with experience in many areas such as machine learning, artificial intelligence, RESTful API's and web development... like this site. Currently I'm working on turning my AI chatbot into a REST service using Django and Flask so the AI itself can be stored in a central location and user's only have to download and run the client.

As a python instructor for Code First Girls, a charity aiming to improve the representation of women and non binary people in the tech industry, I get the oppurtunity to share my passion for programming and pass on my knowledge to people from other fields of study.

Code First Girls

As a volunteer Instructor for Code First Girls I've taught a variety of programming fundamentals such as conditional and logical operators and loops. Students who wished to take their learning further were also taught data visualization using matplotlib.

Cloud Services

Having created an instant messaging REST service hosted on Azure, I've gained a variety of skills crucial for cloud management and creating efficient RESTful API's in Java.

Machine Learning

One of my personal favourite projects was using machine learning models to predict developers income based on a variety of attributes such as their job, years of experience, gender and much more.
